# react-canvas-boilerplate

使用Canvas渲染页面

本项目由 `npx create-react-app --template @liangshen/react-canvas` 命令进行创建的模版项目.

## 环境

node v16+

## 运行

```
npm start
```

## 打包

1.使用npm包引入Layout打包
```
npm run build:lib
```

2.使用了微信小游戏Layout插件打包
```
npm run build:lib:layout-plugin
```

3.也默认提供了浏览器打包
```
npm run build
```
